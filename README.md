# My Linux Desktop Setup

Fully automating of install packages, including system and user set up and enchancing the desktop environment.

- Shell: `oh-my-zsh`
- Interface: `GNOME`
- Terminal: `sakura` and `tmux`

**Tested**

- Ubuntu 18.04 LTS (Pop OS)

**Installation**

```shell
    chmod +x install.sh
    ./install.sh
```
